# 🚀 # Textile Agent: Quality & Sales Intelligence
**Live Demo:** https://textile-quality-sales-agent-813208128293.asia-southeast1.run.app/

This project was developed for the **AI Seekho 2026** program. It uses **Gemini 1.5 Flash** to provide AI-powered quality checklists and marketing summaries for the textile industry.

### Key Features

*   **Fabric Specifications Analysis**: Analyzes heritage textile details to provide instant production standards and technical insights.
*   **Quality & Sales Intelligence**: Uses Gemini 1.5 Flash to generate strategic sales logic and quality benchmarks for factory agents.
*   **Industry-Tailored Logic**: Specifically engineered to handle complex inquiries in the textile sector, bridging the gap between manufacturing and sales.
