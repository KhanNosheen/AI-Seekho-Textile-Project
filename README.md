# 🚀 Textile Quality & Sales Agent
**Live Demo:** https://textile-quality-sales-agent-813208128293.asia-southeast1.run.app/

This project was developed for the **AI Seekho 2026** program. It uses **Gemini 1.5 Flash** to provide AI-powered quality checklists and marketing summaries for the textile industry.

### Features:
* **Quality Checklists:** Instant production standards for items like bedsheets.
* **Sales Summaries:** Professional marketing copy for home decor brands.
* **Textile Focused:** Tailored logic for factory agents and sales leads.
